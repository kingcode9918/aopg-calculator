# Verse Calculator

A calculator for **Verse Piece** on Roblox.

## Features

- **Base Stats Calculator**: Calculate your character's stats with a total cap of 80,000 points
- **Accessory System**: Select accessories with enhancement levels (0-10)
- **Ghost/Rank System**: Assign ranks (D to SSS+) for each stat

## Stats

- **Strength**: Physical damage stat (max 27,500 per stat)
- **Defense**: Damage reduction stat (max 27,500 per stat)
- **Sword**: Sword damage stat (max 27,500 per stat)
- **Special**: Special damage stat (max 27,500 per stat)

## Accessories

Accessories provide stat bonuses that scale with enhancement level:
- Base stats from the accessory
- Additional stats per enhancement level (increment value)

## How to Use

1. Navigate to the Verse Calculator from the home page
2. Set your base stats using the input fields (total max: 80,000)
3. Select a rank for each stat using the dropdown
4. Choose an accessory and set its enhancement level
5. View the calculated bonuses displayed next to each stat
